## 📁 Misato Shimizu | Portfolio Frontend

A minimalist, high-performance portfolio built with Next.js 14 and Tailwind CSS. This project serves as the primary interface for my professional work, featuring a "Business" theme, snap-scroll navigation, and interactive project showcases.

### 🛠️ Tech Stack
- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com) & [daisyUI](https://daisyui.com) (Business Theme)
- **Language:** TypeScript
- **Deployment:** Vercel

### ✨ Key Features
- **Responsive Architecture:** Optimized for all device sizes using Tailwind's mobile-first breakpoints.

- **Dynamic Navigation:** Programmatic ```scrollIntoView``` integration for a seamless single-page experience.

- **Performance Focused:** Leveraging Next.js Server Components (RSC) to minimize client-side JavaScript.

- **Clean UI:** A high-contrast "Business" aesthetic designed for professional readability.

- **Interactive Playground (Upcoming):** A dedicated laboratory section for showcasing experimental features, complex algorithms, and custom API integrations.

### 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**

```bash
git clone https://github.com/misato118/misato-shimizu.git
```

2. **Install dependencies:**

```bash
npm install
```

3. **Run the development server:**

```bash
npm run dev
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 🏗️ Project Structure

```
├── app/            # Next.js App Router (Pages & Layout)
├── components/     # Reusable UI components (Header, Project Cards, etc.)
├── public/         # Static assets (Resume PDF, Images)
├── constants/      # Project data and experience strings
└── tailwind.config.js
```
